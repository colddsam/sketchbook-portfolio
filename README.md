# Samrat | Full Stack Developer & Creative Designer

🎨 **A digital sketchbook where high-fidelity logic meets hand-drawn aesthetics.**

This is my personal portfolio website, built with **Next.js**, **Framer Motion**, and a touch of coffee. It showcases my journey as a "BTech survivor" turned **Full Stack Developer @ Accenture**, blending enterprise-grade Java/Spring Boot experience with creative Graphic Design and Machine Learning.

🚀 **Live at:** [colddsam.com](https://colddsam.com)

## ✨ Features

- **Doodle Aesthetic:** Unique hand-drawn UI style with custom SVG animations.
- **Performance Optimized:** 95%+ reduction in image payloads using modern WebP formats.
- **SEO & Socially Aware:** Dynamic OpenGraph and Twitter images generated on-the-fly (`next/og`).
- **Interactive Experience:** Smooth scrolling (Lenis), tilt cards, and scroll-triggered reveals.
- **PWA Ready:** Full web manifest and multi-platform favicon suite.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16+](https://nextjs.org/) (App Router, Turbopack)
- **Styling:** CSS Modules & Framer Motion for rich animations
- **Logic:** TypeScript, React 19
- **Creative:** Custom hand-drawn SVGs & Canvas-based doodles
- **Backend (Case Studies):** Java, Spring Boot, FastAPI, PostgreSQL

## 📦 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/colddsam/sketchbook-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🚀 Deployment

The site is optimized for deployment on **Vercel**. 

Standard production builds:
```bash
npm run build
```
